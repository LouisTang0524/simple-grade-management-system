# simple-grade-management-system 简易班级成绩管理系统
A simple grade management system as my homework of Advanced Programming Language lesson.  
大一高级语言程序设计大作业，实现一个简单的班级成绩管理系统。

## 语言
C++。

## 如何使用
### STEP 1.
输入所有科目名称及成绩在加权平均成绩中所占权重。
### STEP 2.
按照主菜单和提示文本操作。
### STEP 3.
在主菜单页面输入0可以退出系统。
